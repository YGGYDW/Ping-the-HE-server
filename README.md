# Ping HE 的IPV6隧道服务器

- Windows 脚本测试
- Linux 命令行测试
## Windows 脚本测试
- 两个文件`run_ping1.ps1` `run_tests.bat`
- 双击运行 `run_tests.bat`
- run_ping1.ps1中第5行，可修改Ping的次数：`$Count = 5`
- Ping值结果保存在cvs或xlsx文件
## Linux 命令行测试
- 例如：VPS主机上Debian、Ubuntu中测试，输入以下全部服务器，每个IP服务器Ping4次，命令中可更改次数：`ping -c 5`，复制以下命令到VPS运行:
- `for ip in 216.66.22.2 216.218.200.58 184.105.253.14 184.105.253.10 184.105.250.46 72.52.104.74 64.62.134.130 64.71.156.86 216.66.77.230 66.220.18.42 209.51.161.58 209.51.161.14 66.220.7.82 216.218.226.238 216.66.38.58 184.105.255.26 216.66.86.114 216.66.87.14 216.66.80.30 216.66.87.102 216.66.80.26 216.66.88.98 216.66.84.42 216.66.86.122 216.66.80.90 216.66.80.162 216.66.80.98 216.218.221.6 216.218.221.42 74.82.46.6 216.66.87.98 216.66.87.134 216.66.64.154 216.218.142.50 216.66.90.30; do echo "===== $ip ====="; ping -c 5 $ip; echo; done`
